# Project Description:
"Home Farm" app: a software product designed to organize the activities within a family vegetable farm.
•   The application stores data about the cultivated crops (tomatoes, cucumbers, etc.).
•	Crop-related activities: creating records for planting, spraying, harvesting, and waste/spoilage.
•	Separate records are added for products manufactured from the crops (canned tomatoes, lutenitsa, pickles, etc.).
•	Creation and tracking of customer orders for the crops/products.

# Roles in the App
•	Visitor: views the home page, can register.
•	User: can edit their profile, create orders.
•	Administrator: manages crop-related activities (planting, spraying, harvesting, waste). Manages customer orders. Views and manages users.

# Visitors
Visitors are anonymous and can:
•	Visit the home page.
•	Register in the application (using email and password).

# Users
Registered users in the application have a profile containing a name, password, phone number, and delivery address. They can:
•	Log in and log out of the application.
•	Edit their profile.
•	View their orders and track the order progress (current status).
•	Create orders and add products to them.
•	Cancel a created order (only if the order status is not yet "Accepted").

# Admins
Admins are registered users with extended privileges. They can:
•	Create records for crop-related activities: planting, spraying, harvesting, and waste.
•	Manage orders (change order status).
•	View production statistics (waste, harvest yield, value, sales – both overall and broken down by product).
